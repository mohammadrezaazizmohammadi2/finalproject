## مقدمه

برنامه شبیه ساز دیوار با رابط کاربری اسان و سریع 

##  بخش های مختلف

### حساب کاربری و امنیت
- ثبت‌نام و ورود با شماره تلفن و ایمیل (شناسه یکتاو غیرقابل تغییر حساب)
- احراز هویت مبتنی بر **JWT** (ورود از دستگاه دوم نشست قبلی را باطل می‌کند)
- ثبت نام اولیه با گرفتن دوباره رمز عبور برای جلوگیری از اشتباه
- استفاده از یوزرنیم های یکتا برای کاربران
### آگهی‌ها
- ثبت آگهی با عکس، دسته‌بندی، توضیحات تکمیلی ، شهر و قیمت
- چرخه تأیید: آگهی پس از ثبت در انتظار تأیید ادمین قرار می‌گیرد (PENDING / ACTIVE / REJECTED / SOLD/DELETED)
-  حذف آگهی (حذف با پاپ‌آپ تأیید)، علامت فروخته شد
- جستجو، فیلتر شهر و محدوده قیمت، مرتب‌سازی (جدیدترین / ارزان‌ترین / گران‌ترین)

### علاقه‌مندی‌ها و امتیاز
- افزودن/حذف آگهی به علاقه‌مندی‌ها با آیکون قلب (دکمه هوشمند: وضعیت فعلی را نشان می‌دهد)
- امتیازدهی ۱ تا ۵ ستاره به فروشنده (رای به کاربر است نه اگهی و اگر کسی 10 اگهی داشته باشد فرد دیگر روی هر کدام امتیاز دهد برای همه اگهی ها لحاظ میشود)

### پنل مدیریت
- تأیید / رد آگهی‌ها 
- حذف کاربران یا ارتقای انها به مدیر
- مدیریت دسته‌بندی‌ها  (افزودن/حذف)
- دیدن همه اگهی ها چه حذف شده چه فروخته شده

## خلاصه API

| متد | مسیر | شرح |
|---|---|---|
| POST | `/register` | ثبت‌نام |
| POST | `/login` | ورود (با نام کاربری) و دریافت توکن |
| GET | `/ads` | لیست آگهی‌های تایید شده (با فیلتر و جستجو) |
| GET | `/ads/my` | لیست آگهی‌های خود کاربر (هر وضعیتی) |
| GET | `/ads/all` | لیست همه‌ی آگهی‌ها با هر وضعیتی (فقط ادمین) |
| POST | `/ads` | ثبت آگهی جدید |
| DELETE | `/ads/{id}` | حذف آگهی |
| POST | `/ads/{id}/mark-sold` | علامت‌گذاری آگهی به‌عنوان فروخته‌شده |
| POST | `/ads/{id}/image` | بارگذاری عکس آگهی |
| GET | `/images/{filename}` | دریافت عکس آگهی |
| GET | `/categories` | لیست دسته‌بندی‌ها |
| POST / DELETE | `/categories` ، `/categories/{id}` | افزودن / حذف دسته‌بندی (فقط ادمین) |
| GET | `/admin/ads/pending` | لیست آگهی‌های در انتظار تایید (فقط ادمین) |
| POST | `/admin/ads/{id}/approve` ، `/admin/ads/{id}/reject` | تایید / رد آگهی (فقط ادمین) |
| GET | `/admin/users` | لیست کاربران (فقط ادمین) |
| POST / DELETE | `/admin/users/{id}/promote` ، `/admin/users/{id}` | ارتقا / حذف کاربر (فقط ادمین) |
| GET | `/users/{id}/public` | اطلاعات عمومی کاربر |
| GET | `/users/{id}/rating-summary` | میانگین و تعداد امتیازهای کاربر |
| GET | `/users/{id}/my-rating` | امتیاز قبلی خود کاربر به این فرد |
| POST | `/users/{id}/rate` | ثبت / به‌روزرسانی امتیاز (۱ تا ۵ ستاره) |
